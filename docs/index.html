▸ getClusterData()
src/ExpectationMaximization.js

Returns an array of objects containing all the information related to each cluster (Note: this method is only valid after running the train method). Each element of the array contains:

  • weight: Weight of the current cluster.
  • mean: Current mean of the cluster.
  • covariance: Covariance matrix of the cluster.
  • prediction: prediction label associated with the cluster.
